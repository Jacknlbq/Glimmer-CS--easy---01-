## 题目&解法
1. 高级计算机语言与低级计算机语言，各有什么优劣，你更喜欢哪一类计算机语言？
    >高级语言方便人阅读，更简洁，但计算机不能直接读取,还要进行一些操作。
    >低级语言过于繁琐，但低级语言是计算机可直接读懂的。
    >我肯定更喜欢高级语言啊
